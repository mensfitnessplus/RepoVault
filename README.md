# RepoVault

A mobile-first, Progressive Web App (PWA) designed to be your personal repository and bookmark dashboard. 

## Features
- Mobile-first 2-column grid layout
- Light and Dark themes (auto-saved)
- Automatic Favicon fetching
- Drag-and-drop reordering (via SortableJS)
- Pinned items, Tagging, and filtering
- Full offline support via Service Worker
- LocalStorage architecture (No backend required)
- Import / Export JSON backups

## How to Run
Serve this directory using any static web server (e.g., Live Server, Python's `http.server`, or deploy to GitHub Pages).
